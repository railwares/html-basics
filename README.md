## Що реалізовано

Проєкт складається з трьох взаємопов'язаних сторінок, об'єднаних спільним навігаційним меню.
Було стилізовано палітри кольорів, скидання стандартних відступів, налаштування шрифту та створення центрального контейнера обмеженої ширини. Універсальний стиль для карток контенту (тіні, рамки, заокруглення). Горизонтальне меню за допомогою Flexbox, стани hover та active для посилань, візуальне відокремлення шапки та підвалу від основного контенту. Наявне media query для екранів шириною 768px і менше, перебудова навігації у вертикальний список, гнучка ширина контейнерів та зменшення відступів для мобільних пристроїв.

## Як запустити проєкт
**Варіант 1: Швидкий запуск**
1. Завантажте файли проєкту на свій комп'ютер.
2. Знайдіть файл `index.html` у головній папці.
3. Двічі клікніть по ньому, щоб відкрити у вашому браузері за замовчуванням.

**Варіант 2: Через VS Code**
1. Відкрийте папку проєкту в редакторі Visual Studio Code.
2. Встановіть розширення **Live Server**.
3. Натисніть правою кнопкою миші на файл `index.html` і виберіть `Open with Live Server`.
